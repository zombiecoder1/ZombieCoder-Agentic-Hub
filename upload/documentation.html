📋 পরিচিতি (Introduction)

ZombieCoder API একটি শক্তিশালী AI-চালিত অ্যাপ্লিকেশন যা Next.js 16, TypeScript, এবং Bun Runtime ব্যবহার করে তৈরি। এই সিস্টেমটি স্থানীয়ভাবে AI মডেল (Ollama) এবং ক্লাউড প্রোভাইডারদের সাথে সংযোগ স্থাপন করে রিয়েল-টাইম স্ট্রিমিং চ্যাট সেবা প্রদান করে।

সিস্টেমটির মূল বৈশিষ্ট্য হলো Stock Server - একটি আলাদা Bun-based প্রক্সি সার্ভার যা WebSocket এবং HTTP API এর মাধ্যমে AI প্রোভাইডারদের সাথে সংযোগ পরিচালনা করে।

🎯 মূল উদ্দেশ্য

কোনো mock বা simulation ছাড়াই, সরাসরি AI প্রোভাইডার API-তে কানেক্ট করে সত্যিকারের রিয়েল-টাইম স্ট্রিমিং রেসপন্স প্রদান করা।

🌊 স্ট্রিমিং আর্কিটেকচার (Streaming Architecture)

🔹 কিভাবে স্ট্রিমিং কাজ করে

1

ReadableStream Creation

Next.js ReadableStream API ব্যবহার করে একটি stream তৈরি করা হয় যা chunk-by-chunk data পাঠাতে পারে

2

Fetch with getReader()

AI provider থেকে fetch response-এর response.body.getReader() ব্যবহার করে stream read করা হয়

3

TextDecoder Streaming

TextDecoder ব্যবহার করে binary data UTF-8 text-এ রূপান্তর করা হয়। { stream: true } অপশন দিয়ে partial data handle করা হয়

4

NDJSON Parsing (Ollama)

প্রতি লাইন আলাদা JSON object (Newline Delimited JSON)। parsed.message.content থেকে chunk extract করা হয়

5

SSE Parsing (OpenAI)

Server-Sent Events format: data: {...} লাইনগুলো parse করে choices[0].delta.content extract করা হয়

6

Controller.enqueue()

প্রতিটি chunk controller.enqueue() দিয়ে client-এ পাঠানো হয়। WebSocket হলে ws.send() ব্যবহার করা হয়

🛠️ MCP (Model Context Protocol) আর্কিটেকচার

🔹 MCP কী?

Model Context Protocol (MCP) একটি স্ট্যান্ডার্ডাইজড প্রোটোকল যা AI মডেলদের জন্য tools (ফাংশন) কল করার সুবিধা দেয়। এটি Anthropic-এর তৈরি একটি খোলা স্ট্যান্ডার্ড।

🔹 এই প্রোজেক্টে MCP-এর ব্যবহার

Tool Registration

এজেন্টদের জন্য বিভিন্ন tool (system_info, file_read, file_write) রেজিস্টার করা হয়

Tool Execution

WebSocket বা HTTP এর মাধ্যমে tool কল করা হয় এবং রেজাল্ট পাওয়া যায়

Session Management

প্রতিটি tool execution একটি specific session-এর সাথে যুক্ত

🔹 MCP API Endpoints

Method Endpoint Description
GET /api/mcp/tools সব available tools লিস্ট
POST /api/mcp/execute একটি tool execute করা
GET /api/mcp/tools/:name নির্দিষ্ট tool-এর details
